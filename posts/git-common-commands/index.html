Git Common Commands

Setting Up Git 

1

git config --global user.name "chaosirgit" user.email "qq278935403@email.com";

Create a repository

In the repository directory

1

git init

Adding files to the repository 

1

git add filename;

Commit 

1

git commit -m "desc";

View the current status of the repository 

1

git status

View specific changes 

1

git diff

View Submission History 

1

git log

Rollback to the previous version 

1

git reset --hard HEAD^

HEAD indicates the current version HEAD^ Previous HEAD^^ Previous;

Forward to the new version 

1

git reset --hard commitID

View command history 

1

git reflog

Use to record every command you make

Discard workspace changes 

1

git checkout -- file

Returns the file to the state it was in when it was last committed or added.

Move from staging area to workspace 

1

git reset HEAD file

Move the file file from the staging area after add to the workspace.

Delete from repository 

1

git rm file

Common commands for Github

Create ssh key 

1

ssh-keygen -t rsa -C "278935403@gmail.com"

Generate .ssh directory, id_rsa is private key, can’t leak, id_rsa.pub is public key

Login to github, personal settings, SSH Keys page, fill in the title, copy id_rsa.pub and paste it into the text box to generate the key.

Link local repository with remote repository 

1

git remote add <远程库名> git@github.com:<github账户名>/chaosirgit.git

Push 

1

git push [-u] <远程库名> <分支名>

Use the -u parameter for the first push.

Clone the remote repository 

1

git clone git@github.com:chaosirgit/<仓库名>.git

Git’s branching commands

View branches 

1

git branch

Create Branch 

1

git branch <name>

Checkout Branch 

1

git checkout -b <name>

Merge Branch 

1

git merge <name>

Delete Branch 

1

git branch -d <name>

Resolve conflicts

Resolve the conflict, then commit, and the merge is complete.

View Branch Merge Chart 

1

git log --graph --pretty=oneline --abbrev-commit
