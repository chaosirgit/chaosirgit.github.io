Preface

Laravel’s Eloquent is a great ORM, Eloquent is a standalone module, we can also use Eloquent in our own projects through composer, this article will explain in detail how to integrate Eloquent in your own projects.

Installation

In the project directory, execute

1

composer require illuminate/database:^5.0

Or add it directly to the composer.json file:

 1
 2

{
  "require": {
    "illuminate/database": "^5.0",
    "illuminate/pagination": "^5.0"
  },
  "repositories": {
    "packagist": {
      "type": "composer"
    }
  }
}

This way Eloquent is installed.

            'driver'    => 'mysql',

            'prefix'    => $db_config['tablepre'],
        ];

        $Capsule = new Capsule;
        // Create database link
        $Capsule->addConnection($database);
        // Set global static accessibility
        $Capsule->setAsGlobal();
        // Start Eloquent
        $Capsule->bootEloquent();

This completes the configuration of Eloquent

Use

DB Usage

At the point of use

  • Ensure that the initialization file for Eloquent has been introduced
  • use Illuminate\Database\Capsule\Manager as DB;
    can be used As in: db.php

Model Usage

As long as your model extends Eloquent class, you’re good to go.

use  Illuminate\Database\Eloquent\Model  as Eloquent; 

class User extends  Eloquent 
{
    protected $table = 'users';
}
